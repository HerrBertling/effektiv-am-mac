Globale Shortcuts

Die funktionieren immer, vermutlich sogar in Microsoft-Programmen ;-)

Shortcut Funktion
⌘ + ⇥ Anwendung wechseln
⌘ + < > Fenster innerhalb einer Anwendung wechseln
⌘ + Q Anwendung beenden
⌘ + Z Letzten Arbeitsschritt widerrufen
⇧ + ⌘ + Z Letzten Arbeitsschritt wieder ausführen
⌘ + , Anwendungs-Einstellungen
⌘ + W Fenster/Tab schließen
⇧ + ⌘ + 4 Screenshot-Fadenkreuz öffnen, damit lässt sich ein Screenshot "aufziehen" – besser: Jing

Shortcuts für die Textarbeit

Shortcut Funktion
⇧ + ← / → Einzelne Buchstaben/Ziffern markieren – mehrfacher Druck auf Pfeiltaste: Zusätzliche Buchstaben
⌥ + ← / → Direkt zum nächsten Wort springen (statt einzelne Buchstaben zu durchlaufen)
⇧ + ⌥ + ← / → Ganzes Wort markieren (statt einzelne Buchstaben zu durchlaufen)
⌘ + ← / → Zum Anfang/Ende einer Zeile springen
⌘ + ↑ / ↓ Zum Anfang/Ende eines Dokumentes/Textfeldes springen
⇧ + ↑ / ↓ Ganze Zeile markieren (vertikal)
⇧ + ⌘ + ← / → Ganze Zeile markieren (horizontal, statt einzelne Buchstaben zu durchlaufen)
⌘ + A Gesamten Text markieren
⌘ + C Kopieren
⌘ + V Einfügen
⇧ + ⌥ + ⌘ + V Text ohne Formatierung einfügen

Shortcuts für Mail & Browser

Browser

Shortcut Funktion
⌘ + T Neues Browser-Tab
⇧ + ⌘ + T Gerade geschlossenes Tab öffnen
⌘ + L URL-Zeile markieren
⇧ + ⌘ + I Webseite per Mail verschicken
⌥ + ⌘ + ← / → Tab wechseln nach links oder rechts

Mail

Shortcut Funktion
⌘ + N Neue E-Mail
⌘ + R Antworten
⇧ + ⌘ + R Allen antworten
⌥ + ⌘ + T Mail in den Ordner sortieren, in den zuletzt eine andere Mail sortiert wurde
⇧ + ⌥ + ⌘ + Ö / Ä Einrückungs-Ebene verschieben – nach links (Ö) oder nach rechts (Ä)
